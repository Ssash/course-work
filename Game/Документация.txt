Описание модулей программы.

Используются стандартные бибилиотеки:
iostrem
locale
Windows.h
conio.h

Файл заголовков:
Headerfunction.h

Файл функций:
function.cpp

Глобальные переменные:
gameOver - переменная отвечает за статус игры,в стартовом состоянии она равна false,
после того как мяч залетает за платформу ее состояние меняется на true.
Константы height и width отвечают за размер игрового поля
platformX,platformY - координаты платформы
platform - размер платформы

Для работы программы используются функции находящиеся в файле function.cpp
Определение функций находится в файле Headerfunction.h
startMenu
inputMenu
startGame
draw
platformMove
ballMove
help
topGamer

Назначение функций:
startMenu - выводит на экран меню программы где осуществляется выбор действий пользователя.
Функция принимает один параметр который используется для выбора выводимого на экран изображения.
inputMenu - считывает ввод с клавиатуры,и возвращает значение используемое функцией startMenu.При соблюдении условия функция вызывает другие функции.
startGame - запускает игровой процесс вызывая функции draw,platformMove,ballMove.
draw - выводит на игровое поле платформу и мяч.
platformMove - перемещение платформы по игровому полю.
ballMove - перемещение мяча по игровому полю.
help - описание программы для пользователя(не реализованно).
topGamer - вывод лучших результатов игроков(не реализованно).
